exports(factionData)

Represents a faction in the EVE Online universe. This class holds information about a faction, including its name, description, associated corporation, and statistics such as the number of stations.

Constructor

# new exports(factionData)

Creates an instance of the Faction class.

Parameters:
Name Type Description
factionData Object

The data for the faction.

The unique ID of the faction.

The name of the faction.

The description of the faction.

The ID of the home solar system of the faction.

The ID of the corporation associated with the faction.
